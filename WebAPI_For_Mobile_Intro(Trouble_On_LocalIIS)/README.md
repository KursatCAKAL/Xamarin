<h1>Xamarin ile Web Serviceler</h1>
<h6>Web servisimizi mobil uygulamamızın kullanılabilmesi için öncelikle DB create edip diğer migration ve configurasyonları yapmamız gerekiyor.[WebServiceToDB]<br></h6>
